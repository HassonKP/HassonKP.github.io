About the Game

BlackDog is a third-person action RPG where players explore a grim and mysterious world. Developed as a personal project, this game was a sandbox to experiment with advanced Unreal Engine mechanics, including custom inventory systems, interactive shops, and dynamic combat.

Game Features

Inventory

A custom-designed inventory system for managing and equipping items seamlessly.

Shop

An interactive shop system where players can buy and sell items to enhance gameplay.

Combat

Dynamic combat mechanics with responsive controls and exciting enemy encounters.
